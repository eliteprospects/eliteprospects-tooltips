Eliteprospects Tooltips

What's this?

Enhance your site with informative tooltips about every hockey player you mention in text. All links to a player or staff profile on Eliteprospects.com will automatically get a tooltip.

If you use Wordpress, links can easily be created with this plugin.

Usage

Place this script tag on the page and you are good to go. 

<script src="https://menmo.github.io/eliteprospects-tooltips/eptooltips.min.js"></script>

Links should match one the following patterns. 

http://www.eliteprospects.com/player.php?player=[ID]
http://www.eliteprospects.com/staff.php?staff=[ID]

Bugs?

This script is not heavily tested, and can be influenced by external styles. If you find that the tooltip looks odd on your site feel free to add an issue so that we can take a look. Remember to include a link to your site.

You can also add an issue if you have a feature requests.
